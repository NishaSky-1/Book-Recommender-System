# Book Recommmender System
A Flask-based web application that recommends books to users basedon similarity.

## Feature
- user can select a book from dropdown
- System shows 3 similar book recommendations
- Built using Python, Flask, and Pandas

## Tech Stack
-**Backend**: Python, Flask
-**Frontend**: HTML, Bootstrap
-**Data**: Book.csv, User.csv, Ratings.csv
